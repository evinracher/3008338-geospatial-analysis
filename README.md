## Repository Purpose

This repository contains **academic exercises, workshops, projects, and coursework** for the Geospatial Analysis course, part of the Artificial Intelligence Specialization at Universidad Nacional de Colombia.

### Learning Objectives

- Understand the fundamentals of geospatial data representation and manipulation.
- Apply spatial analysis techniques to real-world geographic datasets.
- Integrate geospatial methods with AI and data science workflows.
- Develop skills in spatial visualization, geostatistics, and geographic information systems (GIS).

### Academic Scope

The repository is **practical and experimental**, combining theoretical foundations of geospatial analysis with hands-on implementation in Python-based geospatial ecosystems.

## How to Run / Reproduce

### 1. Environment Setup

```bash
# Clone the repository
git clone <repository-url>
cd ANALISIS-GEOESPACIAL

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # macOS / Linux
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

> If no `requirements.txt` is provided yet, install common geospatial packages:
>
> ```bash
> pip install numpy pandas geopandas shapely fiona rasterio folium matplotlib seaborn scikit-learn jupyter
> ```

### 3. Run Notebooks

```bash
jupyter notebook
```

Open the desired `.ipynb` file and run cells sequentially.
